# occupalauney

Place points inside your territories on which delaunay will be run on. The triangles generated after your turn belong to you.

## Build instructions

After you have cloned the repo, cd into it and run:

```
mkdir build
cd build
cmake ..
cd ..
cmake --build build
build/bin/occupalauney
```
